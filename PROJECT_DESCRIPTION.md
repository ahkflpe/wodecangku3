# 项目说明文档

## 04 项目核心成果描述

我构建了一个基于AI辅助开发的知名投资机构持仓追踪器桌面应用。它能自动从美国SEC EDGAR系统获取全球39家知名投资机构（巴菲特伯克希尔、桥水基金、高瓴资本、淡马锡等）的13F季度持仓报告，自动解析XML格式数据并存储到本地SQLite数据库，通过PyQt5构建的GUI界面展示持仓详情、支持搜索和导出功能。项目已实现完整的爬虫-解析-存储-展示数据流，支持CSV和Excel格式导出，帮助用户快速分析机构投资动向。

## 功能展示

### 主要功能
- 收录39家全球知名投资机构
- 自动获取13F持仓数据
- 持仓详情查看（股票名称、市值、股数、投票权）
- 按股票名称搜索
- CSV/Excel格式导出

### 技术架构
- Python 3.12 + PyQt5
- SQLite本地存储
- requests + lxml数据获取
- 多线程异步数据获取

### 核心代码模块
- `sec_fetcher.py` - SEC数据获取（支持动态XML文件定位）
- `parser.py` - 13F XML解析
- `database.py` - SQLite数据存储
- `main.py` - PyQt5 GUI界面

## 数据来源

美国证券交易委员会(SEC) EDGAR系统公开13F报告
