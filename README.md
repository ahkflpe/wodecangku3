# 知名投资机构持仓追踪器

## 项目简介

这是一个自动收集和分析全球知名投资机构持仓数据的桌面应用程序。通过美国SEC EDGAR系统的公开13F报告，自动抓取并解析巴菲特、桥水基金、文艺复兴科技等知名投资机构的股票持仓信息。

## 解决的核心痛点

机构投资者每季度提交13F持仓报告，但手动查找和分析数十家机构的数据耗时费力。本工具自动化了这一过程，让用户可以快速：
- 一键获取20+家知名投资机构的最新持仓
- 查看每只股票的市值、股数、投票权等详细信息
- 搜索特定股票被哪些机构持有
- 导出CSV/Excel格式进行进一步分析

## 核心逻辑

1. **数据获取模块 (sec_fetcher.py)**: 通过SEC官方API获取机构13F文件列表，智能定位持仓数据XML文件
2. **数据解析模块 (parser.py)**: 解析XML格式的持仓数据，提取股票名称、CUSIP代码、市值、股数等信息
3. **数据库模块 (database.py)**: SQLite本地存储，支持历史数据追溯
4. **GUI界面 (main.py)**: PyQt5构建的桌面应用，支持机构选择、持仓查看、搜索和导出功能

## 功能特性

- 📊 收录39家全球知名投资机构（包括巴菲特、桥水、高瓴资本、淡马锡等）
-  手动触发更新机构持仓数据
-  按股票名称搜索所有机构持仓
-  支持CSV和Excel格式导出
- 📈 持仓统计和前10大持仓展示
- 🌏 涵盖美国、中国、亚洲投资机构

## 技术栈

- Python 3.12
- PyQt5 (GUI)
- SQLite (数据存储)
- requests (HTTP请求)
- lxml/BeautifulSoup (XML/HTML解析)

## 使用方法

1. 安装依赖: `pip install -r requirements.txt`
2. 运行程序: `python main.py`
3. 选择投资机构，点击"更新数据"获取持仓
4. 查看持仓详情或使用搜索功能
5. 导出CSV/Excel文件

## 数据来源

数据来源于美国证券交易委员会(SEC) EDGAR系统的公开13F报告文件。
